# 更新日志

所有显著的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-12-24

### 新增

- ✨ 支持三种密码类型生成（随机强密码、易记密码、PIN码）
- ✨ 实时密码强度评估系统
- ✨ 密码历史记录功能（本地存储，最多20条）
- ✨ 现代化专业界面设计
- ✨ 响应式布局，完美支持移动端
- ✨ 离线可用（前端包含完整生成逻辑）
- ✨ Docker 容器化部署支持
- ✨ 完整的 API 接口文档
- ✨ Git 提交规范和自动化检查
- ✨ MIT 许可证和贡献指南

### 安全特性

- 🔒 使用加密安全的随机数生成器（Web Crypto API）
- 🔒 API 速率限制（15分钟100次请求）
- 🔒 Helmet 安全中间件保护
- 🔒 可排除易混淆字符选项
- 🔒 所有密码生成均在本地完成，不存储到服务器

### 技术实现

- 🏗️ 基于 Node.js + Express 的后端架构
- 🎨 现代化 CSS3 样式（渐变、动画、Grid/Flexbox）
- 📱 原生 JavaScript ES6+ 实现
- 🐳 Docker 多阶段构建优化
- 🔧 ESLint + Prettier 代码质量控制
- 📝 Commitlint + Husky Git Hooks 提交规范

### 文档

- 📚 完整的使用说明和 API 文档
- 🐳 详细的 Docker 部署指南
- 🤝 贡献指南和行为准则
- ❓ 常见问题解答
